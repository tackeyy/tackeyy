#### 開発中のプロダクト

**株式会社M&Aナビにて**
- **M&Aナビ** — 日本のオンライン M&A マッチングプラットフォーム。
- **ディール管理プラットフォーム** — アドバイザーが M&A 案件を一気通貫で進めるためのクラウドワークスペース。
- **SmartDD** — 生データから投資検討メモを自動起案する、AI 支援の財務デューデリジェンス。

**オープンソース**
- **[mission](https://github.com/tackeyy/mission)** — Claude Code・Codex 向けの品質ゲート付き自律ミッションループ。計画・実行・ピアレビュー・採点を、しきい値を超えるまで反復します。
- **[freee-mcp](https://github.com/freee/freee-mcp)** — freee 公式の会計 API 向け MCP サーバーへのコントリビューター。
- **[ghostty-layout](https://github.com/tackeyy/ghostty-layout)** — Ghostty ターミナルに分割レイアウトを適用する Swift 製 CLI。
- **[xai-cli](https://github.com/tackeyy/xai-cli)** — X の投稿を分析する xAI / Grok 検索 CLI。
- **[slamy](https://github.com/tackeyy/slamy)** — Slack の MCP サーバー兼 CLI。
- **[noty](https://github.com/tackeyy/noty)** — TypeScript 向けの Notion CLI・ライブラリ・MCP サーバー。
- **[google-ads-cli](https://github.com/tackeyy/google-ads-cli)** / **[meta-cli](https://github.com/tackeyy/meta-cli)** — 業務自動化のための広告運用 CLI。

社内向けには、バックオフィス自動化のためのエージェントワークフローを構築しています。Slack 運用、X 運用、会計・税務ワークフロー、監査に耐える内部統制などです。

#### 大切にしていること

AI × M&A · 会計・税務の自動化 · 内部統制と監査証跡 · デモではなく、実際のバックオフィス業務をこなすエージェント。
